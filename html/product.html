遊戲評價
《當個創世神》是一款開放世界遊戲，玩家沒有具體要完成的目標，即玩家在遊戲內有超高的自由度! 遊戲世界由3D方塊組成，並代表不同的材料，如泥土、石頭、各種礦物、水和樹幹。玩家可以在世界各地自由移動， 並收集這些方塊，將其放置在其想要放置的地方，從而進行各項建設。
相關資料
類型 : 沙盒、生存、冒險、解謎、創造
分級 : 普通級
模式 : 單人、多人
開發商 : Mojang
發行日 : 2011年11月18日
遊戲連結 : 官方網站
系統需求
最低需求 :
CPU：Intel Core i3-3210 3.2GHz、AMD A8-7600 APU 3.1GHz或等同性能的CPU
記憶體：4GB
顯示卡：支持 OpenGL 4.4的Nvidia GeForce 400系列或AMD Radeon HD 7000系列
硬碟容量：至少需要1GB用來存儲遊戲核心檔案、世界以及其他檔案
系統： Windows 7及以上
建議需求 :
CPU：Intel Core i5-4690 3.5GHz、AMD A10-7800 APU 3.5GHz或等同性能的CPU
記憶體：8GB
顯示卡：支持OpenGL 4.5的GeForce 700系列或AMD Radeon Rx 200系列
硬碟容量：4-6GB
系統 : Windows 10
